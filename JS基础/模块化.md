#### AMD

* 加载 require.js
* 全局生成 define 函数
* 全局生成 require 函数
* 调用的时候 JS 会自动、异步加载



#### CommonJS

- 一般常用在 node 中，nodejs 模块化规范
- 前端开发依赖的插件和库，都可以从 npm 中获取
- 构建工具的高度自动化，使得使用 npm 的成本非常低
- CommonJS 不会异步加载 JS，而是同步一次性加载进去



---

#### 区别

* 需要异步加载 JS，使用 AMD
* 使用了 npm 之后建议使用 CommonJS
